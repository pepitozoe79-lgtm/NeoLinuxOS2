# 🐧 NeoLinux OS 2 - Premium Web Desktop Environment

![License](https://img.shields.io/badge/License-MIT-6c5ce7?style=for-the-badge)

NeoLinux OS 2 is a state-of-the-art, browser-based Linux desktop simulation. Built with a focus on **premium aesthetics**, **modular architecture**, and **extreme interactivity**, it offers a full OS experience with 50+ integrated applications, all running seamlessly in your browser.

---

## ✨ Features that WOW

### 🖼️ Premium Glassmorphism UI
Experience a stunning interface built with modern **frosted glass effects**, dynamic gradients, and smooth micro-animations. Powered by a custom-built **GlassWM** window manager, NeoLinux OS 2 feels fast, fluid, and premium.

### 📦 50+ Integrated Applications
From productivity to entertainment, NeoLinux comes pre-loaded with everything you need:
- **System**: Terminal (with custom commands), File Manager, System Monitor, Settings.
- **Productivity**: Code Editor, Markdown Editor, Spreadsheet, PDF Viewer, Email Client.
- **Utilities**: Calculator, Unit Converter, Password Generator, QR Code Gen, Dictionary.
- **Creative**: Paint, Image Editor, Color Picker, Photo Viewer.
- **Multimedia**: Music Player, Video Player, Voice Recorder.
- **Games**: Snake Evolution, Tic-Tac-Toe, Minesweeper, Sudoku, Memory Match.

### 💻 Modular Core Architecture
Designed for developers. The entire OS is decoupled into core modules:
- **OS Engine**: Handles system events, taskbar, and the start menu.
- **Window Manager**: Pure JS class for handling multi-window focus, dragging, and resizing.
- **App Registry**: A simple, unified way to register and build new applications.

## 🛠️ Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/pepitozoe79-lgtm/NeoLinuxOS2.git
   ```
2. **Open `index.html` in any modern browser.**
   No installation or dependencies required. Just pure HTML, CSS, and Vanilla JS.

---

## ⌨️ Terminal Commands
Open the terminal and try these:
- `neofetch`: Display system information.
- `help`: List all available commands.
- `cowsay [text]`: Make the penguin talk.
- `fortune`: Get a random piece of wisdom.
